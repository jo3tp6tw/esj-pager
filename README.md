# esj-pager

把 ESJ Zone 章節頁從「長捲動」改成「翻頁閱讀」的 userscript。  
適合電子紙閱讀器，也支援手機與桌面瀏覽器。

## 主要功能

- 全螢幕分頁閱讀（Canvas）
- 正文 + 留言一起閱讀（留言在正文後方）
- 圖片頁支援捲動，超高圖片可上下滑
- 左/右翻頁、滑動翻頁、鍵盤翻頁
- 章節間快速切換（上一章 / 下一章）
- 頁碼跳轉
- 閱讀參數調整（字級、行高、段距、邊距、最大寬度、去除空段落）
- 字體切換（本機字體 / Web Font）
- 記住閱讀進度（上次頁碼）

## 安裝方式

1. 先安裝 userscript 外掛（任一）：
   - Tampermonkey
   - Violentmonkey
2. 匯入 `dist/esj-pager.user.js`
3. 開啟 ESJ 章節頁即可啟用

## 基本操作

- **翻頁**
  - 點擊閱讀區左右側翻頁
  - 滑鼠左鍵按住拖曳可左右翻頁
  - 鍵盤 `← / →` 翻頁
  - 滑鼠右鍵（閱讀區）可上一頁
- **顯示/隱藏工具列**
  - 點擊閱讀區中間
- **全螢幕**
  - 右上角按鈕（展開/縮小）
  - 或按 `F`
- **離開面板/恢復工具列**
  - `Esc`

## 左側選單說明

- **目錄**：回作品目錄
- **閱讀設定**：查看目前參數，開啟調整面板
- **設定檔**：儲存與還原你常用的閱讀參數
- **字體**
  - 字體來源可選：本機字體 / Web Font
  - 本機字體只顯示本機可用選項
  - Web Font 可手動輸入 URL + family，或用一鍵按鈕套用思源字體（可選粗細 100~900）

## 閱讀參數調整

- 面板內調整時會先做局部預覽
- 按「確定」後才完整重算分頁
- 會盡量回到你調整前所在的閱讀位置

## 進度與設定記憶

- 會記住每本小說最近閱讀章節的頁碼
- 會記住你的閱讀參數與字體設定
- 重新開啟同一作品時可延續上次閱讀狀態

## 已知限制

- Web Font 第一次載入可能較慢（視網路而定）
- 本機字體偵測屬前端推估，不是作業系統層級精準查詢
- 目前不提供圖片縮放手勢與獨立圖片檢視器
